# Weather Generator

A simple weather app using OpenWeatherMap API.

## Files
- `index.html` - app UI and weather fetch logic
- `style.css` - styling
- `images/` - icons used by UI

## Setup
1. Open `index.html` in a browser.
2. Enter a city and click search (or press Enter).

## API Key
- The app uses hardcoded key in `index.html`.
- Replace `apikey` with your own from https://openweathermap.org/api for production use.

## GitHub deployment
1. `git init`
2. `git add .`
3. `git commit -m "Initial commit"`
4. `git remote add origin https://github.com/<username>/weather-generator.git`
5. `git branch -M main`
6. `git push -u origin main`

Then configure GitHub Pages in repository settings.
